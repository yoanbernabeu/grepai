## Why grepai?

<CardGrid>
  <Card title="Semantic Search" icon="magnifier">
    Ask in plain English. grepai understands code meaning, not just text patterns.
  </Card>
  <Card title="Call Graph Tracing" icon="random">

## Get started in seconds

<Steps>

1. **Install grepai**

   ```bash
   curl -sSL https://raw.githubusercontent.com/yoanbernabeu/grepai/main/install.sh | sh
   ```

2. **Initialize your project**

   ```bash
   grepai init
   ```

3. **Start indexing & watching**

   ```bash
   grepai watch
   ```

4. **Search with natural language**

   ```bash
   grepai search "user authentication flow"
   ```

5. **Trace function calls**

   ```bash
   grepai trace callers "Login"    # Who calls this?
   grepai trace callees "Handler"  # What does it call?
   ```

</Steps>

## Choose your embedder

<Tabs>
  <TabItem label="Ollama (Local)" icon="laptop">
    ```yaml
    # .grepai/config.yaml
    embedder:
      provider: ollama
      model: nomic-embed-text
    ```
    <Badge text="Recommended" variant="success" /> Privacy-first, runs 100% offline.
  </TabItem>
  <TabItem label="OpenAI (Cloud)" icon="external">
    ```yaml
    # .grepai/config.yaml
    embedder:
      provider: openai
      model: text-embedding-3-small
    ```
    Faster indexing, requires API key.
  </TabItem>
</Tabs>

## Choose your storage

<CardGrid>
  <Card title="GOB — Local File" icon="seti:favicon">
    <Badge text="Default" variant="success" size="small" />

    Zero setup, single portable file. Perfect for small & medium codebases.

    ```yaml
    store:
      backend: gob
    ```
  </Card>
  <Card title="PostgreSQL — pgvector" icon="seti:db">
    <Badge text="Large projects" variant="caution" size="small" />

    Shared index, native vector similarity. Scales to large codebases.

    ```yaml
    store:
      backend: postgres
    ```
  </Card>
</CardGrid>

## Tune your search

<CardGrid>
  <Card title="Search Boost" icon="star">
    <Badge text="Enabled by default" variant="success" size="small" />

    Automatically penalizes test files, mocks, and docs. Boosts source directories.

    ```yaml
    search:
      boost:
        enabled: true
    ```

    <a href="/grepai/search-boost/">Learn more</a>
  </Card>
  <Card title="Hybrid Search" icon="random">
    <Badge text="Optional" variant="note" size="small" />

    Combines vector similarity with text matching (RRF). Better for exact identifiers.

    ```yaml
    search:
      hybrid:
        enabled: true
    ```

    <a href="/grepai/hybrid-search/">Learn more</a>
  </Card>
</CardGrid>
